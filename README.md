# カタログ課題3「＠local」

## 環境
* Ruby 3.0.1
* Rails 6.0.3

## 実行手順
1. 任意のディレクトリにgit cloneする
```
$ git clone git@github.com:erikoenuma/at_local.git
```
2. ディレクトリに移動
```
$ cd at_local
```
3. 以下のコマンドを実行
```
$ bundle install
$ rails db:create
$ rails db:migrate
$ rails db:seed
```
4. サーバーを起動
```
$ rails s
```
5. 以下のURLにアクセスする
```
$ http://localhost:3000/
```
